## Configuration

- Config files can be found in config_files folder. Compose files are provided for both docker and podamn 
- Working .env file in the root. Change credentials and other config if needed. Provided .env is for linux. More details here https://snipe-it.readme.io/docs/configuration
- Generate the app_key and include that in the .env. More details https://snipe-it.readme.io/docs/generate-your-app-key
